mobws
=====

#Project Description
mobws are a set of web services to support different  mobile web clients.
WebApi base web services  
POC to play with different technologies.

#Technologies
* WebApI 
* Dapper (micro ORM)
* SQL Server 2012
* CORS
* MS Test
* Moq

#Features
* Rest base Endpoints
* Supports CORS
* Backend for 2 mobile apps
